
# File Storage App

A full-stack file storage application with **Role-Based Access Control (RBAC)**. Users can create accounts and organizations, upload files, and manage permissions. Admins can invite or remove users from organizations, while regular users can upload and access files based on their permissions.

---

## **Getting Started**

Follow these steps to run the development server locally:

```bash
# Using npm
npm run dev

# Using yarn
yarn dev

# Using pnpm
pnpm dev

# Using bun
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the app running.

You can start editing the application by modifying `app/page.tsx`. The page will **auto-update** as you make changes.

---

## **Features**

* **Role-Based Access Control (RBAC)**: Admins and users have different permissions.
* **Organization Management**: Admins can create organizations, invite users, and remove members.
* **File Management**: Upload files to personal or organization storage.
* **Shared Access**: Users can access files they have permission for.
* **Landing Page**: Custom landing page for navigation and overview.
